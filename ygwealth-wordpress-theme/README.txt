Y&G Wealth WordPress Theme
==========================

Install:
1. In WordPress admin, go to Appearance > Themes > Add New > Upload Theme.
2. Upload ygwealth-wordpress-theme.zip.
3. Activate the theme.

On activation, the theme creates the website pages automatically and sets the homepage.
It also uses clean WordPress URLs such as /about/, /products/, /services/, and /calculators/.

Included pages:
Home, About Us, Products, Product Detail, Insurance, Services, Service Detail,
Calculators, Blog, Blog Detail, Careers, Team, Contact Us, Login, Sign Up,
and Book Appointment.

Notes:
- The theme is self-contained and uses the same content, styling, calculators, menu,
  footer, logos, and lead forms from the static website.
- Product detail, service detail, and blog detail pages use query parameters for the
  selected item, for example /product-detail/?product=mutual-funds.
- Forms are frontend-only, matching the current static website. A backend form plugin
  or CRM integration can be added later if required.
